哎，这后宫的风啊，吹得紧了，连带着这朝堂之上，亦是暗流涌动，不得安宁。今日这Reddit上的喧嚣，一桩桩，一件件，真真假假，虚实难辨，看得本宫头昏眼花。

至于那川普，又在叫嚣着要揭露政府中的“深层势力”？ 呵，“贼喊捉贼”，真是可笑至极。 “欲加之罪，何患无辞”，他口中的“深层势力”，怕是那些不肯与他同流合污，秉持正义良知的忠臣良将吧。 “指鹿为马”，颠倒黑白，这等手段，实在令人不齿。
